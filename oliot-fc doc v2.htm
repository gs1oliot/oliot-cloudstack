Introduction

Oliot-fc is an implementation of EPCglobal application level events (ALE) interface. It provides APIs for filtering and grouping radio frequency identification (RFID) based data. Clients of Oliot-fc can define filtering and grouping pattern according to EPCglobal ALE interface, and the result is returned back to them.

 

 

User guide

Installation

You can install the Oliot-fc server as follows.

1.       Install Java Virtual Machine (JVM)

2.       Install Tomcat

3.       Download oliot-fc web archive file (.war)

4.       Copy oliot-fc web archive file (.war) to Tomcat directory webapps

5.       Start the tomcat

 

Configure Server Using Client

You can configure the Oliot-fc server as follows.

1.       Install Java Virtual Machine

2.       Download oliot-fc client file (.jar)

3.       Prepare ALE specs to use

4.       Run the oliot-fc client

 

 

Developer Guide

This section is for the developers.

 

 

 

Download & Compilation

You have to check prerequisite before you start.

    JVM installed

    Git installed

    Maven installed

 

You can download the source code of Oliot-fc from the Github repository via the git command git clone

    $ git clone https://github.com/gs1oliot/oliot-fc.git

 

You now got the Oliot-fc source code in the subdirectory oliot-fc. Move to the directory.

    cd oliot-fc

 

Oliot-fc is Maven project, so you can compile it with maven command. The following command is for cleaning up binaries, building up new binaries, and installing them to the maven local repository.

    mvn clean install

 

You can check compiled binaries in the following directories.

 

    fc-server/target/fc-server-{version}.war

 

 

Setting up Development Environment Using Eclipse

You can setup the development environment with Eclipse.

1.       Go to the directory oliot-fc

    $ cd oliot-fc

 

2.       Convert Oliot-fc Maven project to Eclipse project

    $ mvn eclipse:eclipse

 

 

3.       Run the Eclipse

 

 

4.       Import as Existing Project
